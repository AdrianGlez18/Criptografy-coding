Cifrado de Vigenere implementado en Go. Para ejecutarlo, hay dos opciones, y ambas necesitan tener go instalado.

1. Interpretarlo directamente.

```bash
go run main.go
```

2. Compilarlo y ejecutarlo.

```bash
go build -o vigenere main.go
chmod +x vigenere
./vigenere
```
